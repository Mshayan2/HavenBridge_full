# HavenBridge Frontend

This folder contains the React + Vite frontend for HavenBridge.

For full project documentation, setup, environment variables, backend integration, and feature overview, see the root README:

- [../README.md](../README.md)

## Frontend Commands

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Frontend Environment

Optional:

```env
VITE_API_BASE=http://localhost:5000/api
```
